# TarGz Library

Go-lang library to tarGz or untarGz stream objects.

## Installation

```bash
go get github.com/TMoynat/targz
```

## Usage

```go
import (
       github.com/TMoynat/targz
)

go_gzip.Ungzipator(stream) // stream is an io.Reader type
```
